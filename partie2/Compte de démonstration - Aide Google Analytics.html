Compte de démonstration

Apprendre en testant les données de Google Merchandise Store et de Flood-It!

Le compte de démonstration est un compte Google Analytics entièrement fonctionnel, accessible à tous les utilisateurs de Google. Il contient une propriété Universal Analytics et deux propriétés Google Analytics 4. Il permet d'examiner des données d'entreprise réelles et de tester les fonctionnalités de Google Analytics.

Accéder au compte de démonstration

Pour accéder au compte de démonstration, cliquez sur le lien ACCÉDER AU COMPTE DE DÉMONSTRATION à la fin de cette section. Voici ce qui se passe lorsque vous cliquez sur le lien :

  • Si vous avez déjà un compte Google, vous êtes invité à vous y connecter.
  • Si ce n'est pas le cas, vous êtes invité à en créer un, puis à vous y connecter.

Lorsque vous cliquez sur le lien ACCÉDER AU COMPTE DE DÉMONSTRATION ci-dessous, vous acceptez que Google effectue l'une ou l'autre des actions suivantes liées à votre compte Google :

  • Si vous disposez déjà d'un compte Google Analytics, nous ajoutons le compte de démonstration à votre compte Analytics.
  • Si vous ne disposez pas encore d'un compte Google Analytics, nous en créons un pour vous, associé à votre compte Google, puis nous ajoutons le compte de démonstration à votre nouveau compte Analytics.

Le compte de démonstration est disponible à partir du sélecteur de compte dans Analytics, qui permet de sélectionner les entreprises et les comptes associés.

Il est comptabilisé dans le nombre maximal de comptes Analytics que vous êtes autorisé à avoir avec un compte Google unique. À l'heure actuelle dans Google Analytics, vous pouvez avoir 100 comptes Analytics maximum par compte Google.

Vous pouvez supprimer le compte de démonstration à tout moment.

Provenance des données

Les données du compte de démonstration Google Analytics proviennent de Google Merchandise Store et de Flood-It!.

Google Merchandise Store est un site d'e-commerce qui vend des produits de marque Google. Il existe à la fois des propriétés Universal Analytics et Google Analytics 4 basées sur le site d'e-commerce. En savoir plus sur les différences entre les types de propriétés

Ces deux propriétés contiennent des données dans le compte qui sont représentatives de ce que vous verriez pour un site d'e-commerce et qui couvrent les types d'informations suivants :

  • Données sur les sources de trafic : informations sur la provenance des visiteurs du site Web (par exemple : données sur le trafic généré par les résultats naturels, par les liens commerciaux, par la publicité display, etc.)
  • Données sur le contenu : informations sur le comportement des utilisateurs sur le site (par exemple : URL des pages consultées par les visiteurs, types d'interactions avec le contenu, etc.)
  • Données transactionnelles : informations sur les transactions enregistrées sur le site Web Google Merchandise Store

Flood-It! est un jeu de réflexion simple mais passionnant dans lequel vous devez inonder l'intégralité du plateau de jeu avec une couleur, en un nombre d'étapes limité. Le jeu est disponible sur Android et iOS. Une propriété Google Analytics 4 existe pour Flood-It!. Elle contient des rapports sur les jeux, et des données provenant de l'application et du site Web qui incluent les informations suivantes :

  • Métriques calculées : informations calculées à partir des métriques existantes telles que le "Revenu moyen par utilisateur actif" et la "Durée d'engagement moyenne". Elles comprennent également les "Conversions réalisées par des acheteurs" et la Tendance périodique des revenus, basées sur les indicateurs clés de performance recommandés de Google Play.
  • Données sur les événements : informations sur l'interaction des utilisateurs avec le contenu. Cela inclut des événements tels que la fin d'un niveau et la réinitialisation d'un niveau.
  • Données d'e-commerce : informations concernant les achats via une application.

Exemples d'utilisation du compte de démonstration

Étant donné que le compte de démonstration affiche des données réelles issues d'un site Web d'e-commerce, il est utile pour explorer les rapports et les fonctionnalités de Google Analytics. Voici quelques actions que vous pouvez effectuer grâce aux deux propriétés Google Analytics 4 :

  • Consulter les rapports Google Analytics 4 standards dans la propriété Merchandise Store
  • Découvrir des rapports sur les jeux dans la propriété Flood-It!
  • Effectuer une analyse de base, par exemple :
    • Déterminer le canal qui permet d'acquérir le plus de nouveaux utilisateurs sur l'ensemble de vos différentes plates-formes
    • Établir le nombre total d'utilisateurs uniques associés à chaque entreprise
    • Comptabiliser le nombre de conversions générées dans l'application Flood-It! et sur le site Web Merchandise Store au cours de la semaine dernière
  • Effectuer une exploration de l'entonnoir : déterminer à quel endroit les utilisateurs entrent dans l'entonnoir de conversion et à quel endroit ils le quittent
  • Effectuer des explorations ponctuelles en glissant-déposant sur une toile plusieurs variables (telles que les différents segments, dimensions et métriques grâce auxquels vous mesurez les performances d'une entreprise) afin de visualiser instantanément vos données

Voici quelques actions que vous pouvez effectuer grâce à la propriété Universal Analytics :

  • Consulter tous les rapports standards contenant des données réelles issues de Google Merchandise Store
  • Afficher les données issues des intégrations Google Ads et Search Console
  • Modifier les rapports en ajoutant des filtres et des dimensions secondaires, et en changeant la vue de rapports
  • Comparer les plages de dates en termes de données sur l'acquisition, le comportement et les conversions
  • Segmenter les données en utilisant vos propres segments personnalisés
  • Créer vos propres tableaux de bord, rapports personnalisés et modèles d'attribution
  • Afficher des tableaux de bord et des segments prédéfinis importés à partir de la Galerie de solutions

Limites

Tous les utilisateurs disposent du rôle Lecteur pour le compte de démonstration Google Analytics. Vous pouvez ainsi :

  • consulter les données de rapport et les paramètres de configuration ;
  • manipuler des données dans les rapports (par exemple, filtrer un tableau, ajouter une dimension secondaire, créer un segment) ;
  • créer des éléments personnels tels que des rapports personnalisés, les partager et consulter les éléments partagés. La collaboration sur les éléments partagés n'est pas autorisée.

Pour en savoir plus sur les autorisations de l'utilisateur, consultez la section Gestion des utilisateurs de ce centre d'aide.

Les données du compte de démonstration sont obscurcies, mais correspondent toujours à des données Analytics types.

La propriété Universal Analytics ne comprend pas le rapport Explorateur d'utilisateurs.

Les propriétés Google Analytics 4 n'incluent pas de fonctionnalité d'exportation des rapports ou des explorations, la possibilité d'afficher la dimension "ID de l'appareil" ni l'utilisation de la technique "Explorateur d'utilisateurs" dans l'espace de travail Explorations.

Vous ne pouvez pas utiliser le compte de démonstration avec l'API Reporting d'Analytics (cela est vrai pour les deux types de propriété). Si vous tentez d'y accéder via cette API, vous rencontrerez une erreur d'autorisation.

Supprimer l'accès au compte de démonstration

Pour supprimer votre accès au compte de démonstration, suivez la procédure ci-dessous. Vous pourrez toujours récupérer l'accès ultérieurement.

  1. Connectez-vous à Google Analytics.
  2. Cliquez sur Administration.
  3. Dans le menu COMPTE, sélectionnez Compte de démonstration.
  4. Dans la colonne COMPTE, cliquez sur Gestion des utilisateurs.
  5. Cliquez sur ME SUPPRIMER.
